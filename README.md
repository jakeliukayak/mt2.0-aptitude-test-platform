# Aptitude Test Platform

A comprehensive web-based aptitude test platform featuring multiple reasoning test modes with timer functionality and detailed scoring.

## Features

- **5 Test Modes**:
  - Numerical Reasoning (2 questions)
  - Verbal Reasoning (2 questions)
  - Inductive Reasoning (2 questions)
  - Deductive Reasoning (2 questions)
  - Combined Test (8 questions - 2 from each category)

- **Question Bank**: 10 unique questions for each reasoning category (40 total questions)
- **Timer**: 1 minute per question with visual countdown and warnings
- **Scoring System**: 
  - Percentage and fraction display
  - Category breakdown for combined tests
  - Performance-based feedback messages
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Material Design**: Google-inspired UI with smooth animations

## Usage

Simply open `index.html` in your web browser or visit the GitHub Pages deployment.

1. Select a test mode from the home screen
2. Answer each question within the 1-minute time limit
3. View your results and category breakdown at the end

## GitHub Pages Deployment

This application is ready to be hosted on GitHub Pages. Simply enable GitHub Pages in your repository settings and point it to the main branch.
